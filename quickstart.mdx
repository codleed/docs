## Prerequisites

Before you begin, make sure you have the following installed:

- **Node.js** v20 or later
- **pnpm** v10+ (`npm i -g pnpm`)
- **Docker** and **Docker Compose** (for PostgreSQL, Redis, and MinIO)

## Step 1: Clone and install

```bash
git clone https://github.com/Codleed/learningopk.git
cd learningopk/learningopk
pnpm install
```

## Step 2: Start infrastructure services

Spin up PostgreSQL 16, Redis 7, MinIO, PgBouncer, and Nginx with a single command:

```bash
pnpm docker:up
```

<Note>
  PostgreSQL is exposed on host port **5433** to avoid collisions with any existing local PostgreSQL instance on port 5432.
</Note>

Verify all containers are running:

```bash
docker ps
```

You should see containers for `learningopk_postgres`, `learningopk_redis`, `learningopk_minio`, `learningopk_pgbouncer`, and `learningopk_nginx`.

## Step 3: Configure environment variables

Create environment files from the provided templates:

```bash
# Backend
cp backend/.env.example backend/.env

# Frontend
cp frontend/.env.local.example frontend/.env.local
```

Edit `backend/.env` and set the following required values:

```bash
# Database (connects via PgBouncer)
DATABASE_URL=postgresql://postgres:password@localhost:6432/learningo

# Redis
REDIS_URL=redis://localhost:6379

# Better Auth
BETTER_AUTH_SECRET=<generate-a-random-32-char-string>
BETTER_AUTH_URL=http://localhost:3001
FRONTEND_ORIGIN=http://localhost:3000

# AI — get a free key from https://console.mistral.ai
MISTRAL_API_KEY=<your-mistral-api-key>
```

<Tip>
  The Mistral AI API offers a free tier with generous limits. Sign up at [console.mistral.ai](https://console.mistral.ai) to get your API key.
</Tip>

## Step 4: Run database migrations

Apply the Drizzle schema to your local PostgreSQL instance:

```bash
pnpm db:migrate
```

To seed sample content (chapters, quizzes, flashcards):

```bash
pnpm db:seed
```

## Step 5: Start development servers

Start both frontend and backend in parallel:

```bash
pnpm dev:all
```

Or start them individually:

```bash
pnpm dev:backend   # Express API on http://localhost:3001
pnpm dev:frontend  # Next.js app on http://localhost:3000
```

## Step 6: Open in browser

Navigate to [http://localhost:3000](http://localhost:3000) to see the LearningoPK platform.

## Available Scripts

| Command | Description |
|---------|-------------|
| `pnpm dev:all` | Start frontend + backend concurrently |
| `pnpm dev:frontend` | Start Next.js dev server (port 3000) |
| `pnpm dev:backend` | Start Express API server (port 3001) |
| `pnpm docker:up` | Start PostgreSQL, PgBouncer, Redis, MinIO, Nginx containers |
| `pnpm docker:down` | Stop all containers |
| `pnpm db:generate` | Generate Drizzle migrations |
| `pnpm db:migrate` | Run Drizzle migrations |
| `pnpm db:seed` | Seed database with sample content |
| `pnpm db:studio` | Open Drizzle Studio (visual DB browser) |
| `pnpm db:push` | Push schema directly (dev only) |
| `pnpm db:clear` | Clear all database tables |
| `pnpm lint` | Lint all packages |
| `pnpm typecheck` | TypeScript check all packages |
| `pnpm verify:services` | Health check all services |
