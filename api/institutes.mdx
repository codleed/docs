## Overview

The Institutes API returns all educational institutes registered in the system. Institutes are ordered alphabetically by name.

<Warning>
This router is defined in `backend/src/routes/institutes.ts` but is **not currently mounted** in `server.ts`. The institutes data is available through the admin content endpoints.
</Warning>

---

## List institutes

Returns all institutes ordered alphabetically by name.

```bash
GET /api/institutes
```

**Auth**: None (public endpoint)

### Example

<CodeGroup>

```bash cURL
curl http://localhost:3001/api/institutes
```

</CodeGroup>

### Response

**Status**: `200 OK`

```json
{
  "institutes": [
    {
      "id": 1,
      "name": "Cadet College Hasan Abdal"
    },
    {
      "id": 2,
      "name": "Punjab College"
    }
  ]
}
```

<ResponseField name="institutes" type="array" required>
  Array of institute objects.

  <Expandable title="institute object">
    <ResponseField name="id" type="number" required>
      Unique institute identifier.
    </ResponseField>

    <ResponseField name="name" type="string" required>
      Display name of the institute.
    </ResponseField>
  </Expandable>
</ResponseField>

### Error responses

| Status | Condition |
|--------|-----------|
| `500` | Database error — failed to fetch institutes |
