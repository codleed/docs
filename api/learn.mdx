## Overview

The Learning API serves curriculum content organized by board, grade, subject, and chapter. All content endpoints use slug-based URL routing (`/:board/:grade/:subject/:chapter`).

---

## Get subject with chapters

Retrieve a subject and its chapter list by board, grade, and subject slugs.

```bash
GET /api/learn/:board/:grade/:subject
```

**Auth**: None

### Path parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `board` | string | Board slug (e.g., `punjab`, `federal`) — lowercase alphanumeric with hyphens |
| `grade` | string | Grade slug (e.g., `9`, `10`) — lowercase alphanumeric with hyphens |
| `subject` | string | Subject slug (e.g., `physics`, `chemistry`) — lowercase alphanumeric with hyphens |

### Response

```json
{
  "board": {
    "slug": "punjab",
    "name": "Punjab Board"
  },
  "grade": "9",
  "class": {
    "slug": "9",
    "name": "9th"
  },
  "subject": {
    "id": 1,
    "slug": "physics",
    "name": "Physics",
    "description": "Introduction to physics concepts for class 9"
  },
  "chapters": [
    {
      "id": 1,
      "chapterNumber": 1,
      "title": "Physical Quantities and Measurement",
      "slug": "physical-quantities-and-measurement",
      "hasExercises": true,
      "hasQuiz": true
    }
  ]
}
```

### Error responses

| Status | Condition |
|--------|-----------|
| `400` | Invalid route parameters (slug format mismatch) |
| `404` | Subject not found |

## Get chapter progress graph

Retrieve a graph of chapters with progress status for the authenticated user. Students can only access their own board/class combination.

```bash
GET /api/learn/:board/:grade/:subject/graph
```

**Auth**: Student

### Path parameters

Same as [Get subject with chapters](#get-subject-with-chapters).

### Response

```json
{
  "graph": {
    "nodes": [
      {
        "id": 1,
        "title": "Physical Quantities and Measurement",
        "progressPercent": 75
      }
    ],
    "edges": [
      { "source": 1, "target": 2 }
    ]
  }
}
```

### Error responses

| Status | Condition |
|--------|-----------|
| `400` | Invalid route parameters |
| `403` | Forbidden — student's board or class does not match the requested route |
| `404` | Subject not found |

<Note>
Students are restricted to their registered board and class. Admins can access any combination.
</Note>

## Get chapter detail

Retrieve a single chapter with its exercises, flashcards, and quiz metadata.

```bash
GET /api/learn/:board/:grade/:subject/:chapter
```

**Auth**: None

### Path parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `board` | string | Board slug |
| `grade` | string | Grade slug |
| `subject` | string | Subject slug |
| `chapter` | string | Chapter slug (e.g., `physical-quantities-and-measurement`) |

### Response

    "name": "Punjab Board"

      "id": 1,
      "exerciseNumber": "1.1",
      "question": "Define physical quantities.",
      "solution": "Physical quantities are...",
      "difficulty": "easy",

## Get Exam Patterns

Retrieve exam pattern analysis for a subject, showing which chapters frequently appear in exams.

```bash
GET /api/learn/patterns/:board/:subject
```

**Auth**: None

### Path parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `board` | string | Board slug |
| `subject` | string | Subject slug |

### Response

```json
{
  "success": true,
  "data": {
    "board": "punjab",
    "subject": "physics",
    "patterns": [
      {
        "chapterId": 1,
        "chapterTitle": "Physical Quantities",
        "weightage": 15,
        "frequency": "very_high",
        "topics": ["measurement", "units", "scalars", "vectors"]
      },
      {
        "chapterId": 5,
        "chapterTitle": "Newton's Laws",
        "weightage": 20,
        "frequency": "high",
        "topics": ["force", "acceleration", "friction", "tension"]
      }
    ],
    "summary": {
      "totalChapters": 10,
      "coveredChapters": 8,
      "averageWeightage": 12.5
    }
  }
}
```

### Weightage levels

| Level | Description |
|-------|-------------|
| `very_high` | 15%+ exam weight |
| `high` | 10-15% exam weight |
| `medium` | 5-10% exam weight |
| `low` | <5% exam weight |

## Get Personalized Revision Notes

Generate AI-powered personalized revision notes for a chapter based on the student's weak areas.

```bash
POST /api/learn/revision/:chapterId/personalize
```

**Auth**: Student

### Path parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `chapterId` | number | Chapter ID |

### Request body

| Field | Type | Description |
|-------|------|-------------|
| `focusAreas` | string[]? | Specific topics to focus on (optional) |

### Response

```json
{
  "success": true,
  "data": {
    "chapterId": 5,
    "revisionNotes": "## Key Concepts\n\n### Newton's First Law (Inertia)\nAn object at rest stays at rest...",
    "focusedTopics": ["force", "acceleration"],
    "generatedAt": "2026-04-10T10:00:00.000Z"
  }
}
```

### Notes
- Uses student's AI context (weak/strong topics) to personalize
- Returns markdown-formatted revision notes
- Cached for 1 hour to avoid repeated AI calls
