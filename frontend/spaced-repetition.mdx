## Overview

The Spaced Repetition System (SRS) helps students memorize content efficiently using the SM-2 algorithm. Students review flashcards at optimal intervals to maximize retention with minimal effort.

## User Interactions

### Starting a Review Session

1. Click "Review Now" on dashboard widget
2. Or navigate to chapter's flashcard tab
3. Review due cards one by one

### Rating a Card

After seeing the answer, rate your recall:

| Rating | Meaning | Next Review |
|--------|---------|-------------|
| **Again** | Complete blackout | 1 minute |
| **Hard** | Recalled with difficulty | Short interval |
| **Good** | Recalled correctly | Normal interval |
| **Easy** | Effortless recall | Long interval |

### Viewing Progress

- Stats page shows total reviewed, due, and mastered
- Streak shows consecutive days of reviews

3. Statistics update → Fetch via `GET /api/flashcard-reviews/stats`

## SM-2 Algorithm

The system uses a modified SM-2 algorithm:

### Initial Values
- `interval`: 0 days
- `easeFactor`: 2.5
- `repetitions`: 0

### On Review

```typescript
if (rating === 'again') {
  interval = 0; // Review again in 1 minute
  easeFactor = Math.max(1.3, easeFactor - 0.2);
  repetitions = 0;
} else if (rating === 'hard') {
  interval = interval * 1.2;
  easeFactor = easeFactor - 0.15;
} else if (rating === 'good') {
  interval = interval * easeFactor;
} else if (rating === 'easy') {
  interval = interval * easeFactor * 1.3;
  easeFactor = easeFactor + 0.15;
}
repetitions++;
```

### Mastery Levels

| Level | Interval | Description |
|-------|----------|-------------|
| New | 0 | Never reviewed |
| Learning | 1-6 days | Early reviews |
| Young | 7-21 days | Building retention |
| Mastered | 21+ days | Long-term memory |
