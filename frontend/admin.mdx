## Overview

The admin panel provides tools for managing curriculum content, users, and platform analytics. Accessible only to users with the `admin` role.

## Sections

### Command Center

- **Overview** — High-level KPIs, moderation alerts, recent admin activity
- **Analytics** — Active students, quiz attempts, average scores, subject performance

### Content Management

- **Boards** — Create, edit, delete education boards
- **Classes** — Create, edit, delete classes within boards
- **Subjects** — Create, edit, delete subjects per class
- **Chapters** — Create, edit, publish/unpublish, rename chapters with wiki-link management
- **Exercises** — Add exercises (mcq/short/long/numerical/fill_in_blanks) with solutions
- **Quizzes** — Build quizzes with MCQ questions, manage quiz metadata
- **Quiz Questions** — Add, edit, delete questions per quiz
- **Flashcards** — Create, edit, delete, reorder flashcards per chapter

### User Management

- View all registered users with search, role, and status filters
- Update user roles (promote/demote)
- Suspend/reactivate users with reason tracking

### Community Moderation

- View and pin/unpin forum threads
- Resolve moderation flags with notes
- Admin thread view with solved/pinned/flag filters

### Notifications

- Create broadcast notifications (all/students/admins)
- View notification history

### Settings

- Update platform settings: forum auto-lock hours, quiz pass threshold, maintenance banner

### Audit Logs

- Content, forum, moderation, users, notifications, and settings audit trails
- Aggregated view with scope, status, and search filters

### Jobs

- View queue stats (waiting, active, completed, failed) per queue
- Retry failed jobs

## Access Control

The admin panel route is protected by role-based access control. Non-admin users are redirected to the student dashboard.

```
/admin/* → Requires role: admin
```
