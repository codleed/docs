## Overview

The LearningoPK frontend is built with **Next.js 16** using the App Router. It uses Server Components for data fetching and Client Components for interactive features.

## Technology Stack

- **Next.js 16** — App Router with RSC
- **Tailwind CSS 4** — Utility-first styling
- **HeroUI** — Accessible component library
- **Vercel AI SDK** — `useChat` for AI streaming
- **React Hook Form + Zod** — Form handling and validation

## Directory Structure

```

├── app/                    # App Router pages and layouts
│   ├── (auth)/             # Auth pages (login, register)
│   ├── (dashboard)/        # Student dashboard, subjects, stats, calendar, settings
│   ├── (learn)/            # Learning pages: [board]/[grade]/[subject]/[chapter], past-papers
│   ├── admin/              # Admin panel (17 sections)
│   ├── ai-tutor/           # AI chat interface
│   ├── forum/              # Community forum
│   ├── design-system-demo/ # Design system showcase
│   ├── layout.tsx          # Root layout
│   └── page.tsx            # Landing page
├── src/
│   ├── components/
│   │   ├── ui/             # 24 shadcn-style primitives
│   │   ├── admin/          # 50 admin components
│   │   ├── ai/             # AI tutor + unified chat
│   │   ├── auth/           # 11 auth components
│   │   ├── common/         # 12 shared components
│   │   ├── dashboard/      # 10 dashboard components
│   │   ├── forum/          # 12 forum components
│   │   ├── foundation/     # 9 foundation components
│   │   ├── gamification/   # 7 gamification components
│   │   ├── learn/          # 33 learning components
│   │   ├── stats/          # 15 stats components
│   │   └── settings/       # Settings components
│   ├── hooks/              # Custom React hooks
│   ├── lib/                # 19 utility functions
│   ├── design-system/      # Legacy (deprecated, re-exports ui/)
│   └── styles/             # Global styles
├── tests/e2e/              # 25 Playwright E2E specs
└── public/                 # Static assets
```

## UI Components

Reusable components are in `src/components/ui/` and built with **CVA** (Class Variance Authority). They follow a 4px spacing grid and support dark theme.

```typescript
import { Button, Card, Badge } from "@/components/ui";
```

## API Communication

The frontend communicates with the Express backend via `/api/*` endpoints. The Next.js development server proxies API requests to the backend on port 3001.
