## Overview

The dashboard is the student's home page after login. It provides a snapshot of their learning progress, recent activity, and quick access to continue studying.

<img src="/screenshots/dashboard.png" alt="Student Dashboard" style={{ borderRadius: '8px', border: '1px solid #e5e7eb' }} />

## Components

### Subject Progress Cards

Displays each enrolled subject with:

- Chapter completion percentage
- Average quiz score
- Last studied chapter

### Recent Activity Feed

Shows the latest actions:

- Chapters visited
- Quiz attempts and scores
- AI tutor sessions

### Quick Actions

- Continue where you left off
- Start a new quiz
- Open AI tutor
- Visit the forum

## Data Sources

The dashboard fetches data from:

- `GET /api/progress/stats` — Aggregated statistics
- `GET /api/progress` — Per-chapter progress
- `GET /api/boards` and `GET /api/subjects` — Curriculum structure
