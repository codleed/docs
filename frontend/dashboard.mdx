## Overview

The dashboard is the student's home page after login. It provides a snapshot of their learning progress, recent activity, and quick access to continue studying.

## Components

### DashboardClient

Main client component that orchestrates all dashboard sections.

### Subject Progress Cards

Displays each enrolled subject with:

- Chapter completion percentage
- Average quiz score
- Last studied chapter

### Recent Activity Feed

Shows the latest actions:

- Chapters visited
- Quiz attempts and scores
- AI tutor sessions

### Streak Card

Displays current study streak with streak freeze status.

### Welcome Card

Personalized greeting with student name and XP info.

### StatsCards

Quick stats overview with KPI numbers.

### Weekly Activity Heatmap

GitHub-style calendar showing study intensity per day.

### Dashboard Chrome Layout

Full dashboard layout with settings panel.

## Data Sources

The dashboard fetches data from:

- `GET /api/progress/dashboard` — Full dashboard: streaks, subjects summary, recent activity, quiz history, XP, activity calendar
- `GET /api/progress/dashboard/:boardSlug/:grade/:subjectSlug` — Subject-level dashboard with chapter progress
- `GET /api/boards` — Board list
- `GET /api/subjects` — Subject list
