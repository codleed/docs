## What is LearningoPK?

LearningoPK is a full-stack web learning platform built for 9th and 10th grade students in Pakistan. It provides chapter-wise solutions, an AI-powered Socratic tutor, quizzes, mock exams, flashcards, a student community forum, and subject-wise progress tracking.

The platform covers three major boards:

- **Federal Board** — All core and elective subjects
- **Punjab Board** — All core and elective subjects
- **Sindh Board** — All core and elective subjects

## Tech Stack at a Glance

| Layer | Technology |
|-------|-----------|
| **Frontend** | Next.js 16 (App Router), Tailwind CSS 4, shadcn/ui |
| **Backend** | Express.js 5, TypeScript |
| **Database** | PostgreSQL 16 (Docker) |
| **ORM** | Drizzle ORM (22 tables) |
| **Auth** | Better Auth with Drizzle adapter |
| **AI** | Mistral AI via Vercel AI SDK |
| **Cache** | Redis 7 |
| **Storage** | MinIO (S3-compatible) |
| **Testing** | Node.js test runner + Playwright E2E |
| **Package Manager** | pnpm workspaces (monorepo) |
